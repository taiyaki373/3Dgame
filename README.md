# 3D Game

シンプルな3Dゲーム。広いフィールドにプレイヤーがいるだけのシーン。

## セットアップ

```bash
npm install
npm start
```

ブラウザで `http://localhost:8080` を開いてゲームをプレイできます。

## 操作方法

- **WASD** - プレイヤーの移動
- **マウス** - カメラの視点操作（画面クリックでマウスロック）

## 技術スタック

- **Three.js** - 3D描画
- **JavaScript** - ゲームロジック

## 機能

- 500x500のグリーンな地面
- プレイヤーキャラクター（赤いカプセル形状）
- 三人称カメラビュー
- 滑らかなプレイヤー移動
- 影の描画
